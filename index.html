Ferramenta de Estudo - Conceitos de Programação

Responda as perguntas abaixo e marque se sua resposta está correta!

1. O que é uma variável?

Uma variável é um espaço de armazenamento usado para guardar dados durante a execução do programa.

2. O que é uma função?

Uma função é um bloco de código que realiza uma tarefa específica e pode ser reutilizado em várias partes do programa.

3. O que é um loop?

Um loop é uma estrutura que repete um bloco de código enquanto uma condição for verdadeira.

4. O que é uma condição (if)?

A condição "if" permite que o programa execute um bloco de código apenas se uma condição for verdadeira.

Você acertou 0 de 4!
